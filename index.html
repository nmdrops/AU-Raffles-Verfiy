Independent Draw Verification

How it works

When a raffle is created, the bot generates a secret seed and publishes its SHA-256 commit hash in the announcement. This proves the outcome was predetermined. After the draw, the seed is revealed. Anyone can paste the values below and verify the result was not manipulated.

📍 Where to find these values

🔐 Commit Hash
Posted in #active-raffles when the raffle is created — look for the "Provably Fair" field in the announcement embed.
🌱 Revealed Seed
Shown in the #active-raffles closed embed after the draw has been run, under "Seed Revealed".
🏆 Number of Prizes
Listed in the original raffle announcement under "Number of Prizes".
📋 Entry List JSON
Run /verify <raffle_id> in the Discord server — the bot will show you the full entry snapshot to paste here. The Raffle ID is in the footer of every raffle announcement.
